# UIGen

A UI component generator app built with Next.js and AI.

## Setup

1. Install dependencies:
```
npm run setup
```

2. Start the app:
```
set NODE_OPTIONS=--require ./node-compat.cjs && npx next dev --turbopack
```

3. Open http://localhost:3000

## Features

- Generate React components with AI
- Live preview
- Code editor
- Authentication system
